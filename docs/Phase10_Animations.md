# Этап 10 — Lottie/анимации на Splash/Login/EmptyState (сборка 100%)

## Цель этапа

Добавление современных векторных анимаций для улучшения визуального отклика приложения. Реализация Splash-экрана (заставки) и обновление окна входа с использованием технологии Lottie.

## Файлы (создать/изменить)

1.  **ClinicApp.Wpf/Assets/loading.json** — Файл анимации Lottie (JSON).
2.  **ClinicApp.Wpf/Views/SplashView.xaml** — Новый экран заставки с анимацией и прогресс-баром.
3.  **ClinicApp.Wpf/ViewModels/SplashViewModel.cs** — Логика задержки и автоматического перехода к логину.
4.  **ClinicApp.Wpf/Views/LoginView.xaml** — Обновление: замена статической иконки на Lottie-анимацию.
5.  **ClinicApp.Wpf/ViewModels/MainViewModel.cs** — Обновление: установка SplashView как стартового экрана.
6.  **ClinicApp.Wpf/App.xaml.cs** — Регистрация SplashViewModel в DI.
7.  **ClinicApp.Wpf/ClinicApp.Wpf.csproj** — Добавление зависимости `LottieSharp` и включение ресурсов Assets.

## Технические детали

*   **Библиотека:** `LottieSharp` (версия 2.4.0) для рендеринга After Effects анимаций в WPF.
*   **Ресурсы:** Анимации включены в проект как `Resource`, что позволяет упаковывать их внутрь .exe файла.
*   **Splash Screen:** При запуске приложение показывает заставку на 3 секунды, имитируя загрузку базы данных и ресурсов, после чего перенаправляет пользователя на окно входа.

## Действия в Visual Studio

1.  Открыть `ClinicApp.sln`.
2.  Нажать `Ctrl+Shift+B` для сборки.
3.  Нажать `F5` для запуска.
4.  При запуске вы увидите анимированную заставку ClinicApp.
