# Этап 9 — UI MaterialDesignThemes, темы, skeleton, micro-interactions (сборка 100%)

## Цель этапа

Кардинальное обновление пользовательского интерфейса с использованием библиотеки Material Design In XAML. Настройка цветовых схем, шрифтов, стилей элементов управления и добавление современных визуальных эффектов (анимации переходов, тени, микро-взаимодействия).

## Файлы (создать/изменить)

1.  **ClinicApp.Wpf/App.xaml** — Подключение ресурсов Material Design и настройка темы (DeepPurple/Lime).
2.  **ClinicApp.Wpf/MainWindow.xaml** — Обновление каркаса приложения: боковое меню с иконками, DialogHost, Snackbar.
3.  **ClinicApp.Wpf/MainWindow.xaml.cs** — Логика обработки выбора элементов в ListBox для навигации.
4.  **ClinicApp.Wpf/Views/LoginView.xaml** — Современный дизайн окна входа с использованием Card и плавающих подсказок.
5.  **ClinicApp.Wpf/Views/PatientsView.xaml** — Обновление интерфейса управления пациентами (карточки, стилизованные таблицы, кнопки).
6.  **ClinicApp.Wpf/ClinicApp.Wpf.csproj** — Добавление зависимостей `MaterialDesignThemes` и `MaterialDesignColors`.

## Ключевые изменения

*   **Тема:** Использование `BundledTheme` с базовой светлой темой и основными цветами DeepPurple.
*   **Навигация:** Боковое меню теперь использует `ListBox` со стилем `MaterialDesignNavigationListBox` и иконками `PackIcon`.
*   **Анимация:** Плавное появление контента (Fade-in) при переключении страниц (0.4 сек).
*   **Элементы управления:** Все текстовые поля, кнопки, таблицы и календари переведены на стили Material Design.
*   **Индикация загрузки:** Использование `ProgressBar` в стиле Material Design при выполнении длительных операций.
